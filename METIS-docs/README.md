# METIS User Docs

This folder is the landing zone for user-facing METIS product documentation (as distinct from
`METIS-api/`, which is the API reference for developers).

Drop new `.md` files anywhere under here — the docs site build picks up this whole folder
automatically, no workflow changes needed. Add new pages to the `nav:` section of `mkdocs.yml`
so they show up in the site's sidebar.
